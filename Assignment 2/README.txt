Steps:
1. Make a new project on Google developers console using gmail id.
2. Install ODK aggregate with required fields  (project ID, project path, username)
3. Sign-in using the above Gmail ID and upload ODK Aggregate to Google App Engine.
4. Download ODK collect. 
5. Check project ID.appspot.com to verify that ODK aggregate has been successfully uploaded.
6. Configure ODK collect with username and password and project ID.appspot.com
7. Create XML form using ODK build and download it. 
8. Upload XML form to ODK aggregate on project ID.appspot.com
9. Get blank form on ODK collect and fill it.
10. Finalise form and send it to server.
11. Check server for response.
